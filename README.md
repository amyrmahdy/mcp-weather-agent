# MCP Weather Agent

Agentic weather assistant demonstrating MCP (Model Context Protocol), cloud deployment, and MLOps practices.

## Skills Demonstrated
- **MCP/Agentic Frameworks**: Custom MCP server with tool discovery
- **Cloud**: Deployment-ready architecture
- **CI/CD & MLOps**: MLflow tracking, automated pipelines

## Stack
- Python 3.12
- MCP SDK
- MLflow
- Open-Meteo API

## Setup
```bash
uv sync
```

## Development
Built incrementally via feature branches merged to `dev`, then to `master`.
